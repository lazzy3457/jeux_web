# Le jeu racup des bits

## le but du jeu 

tu est un robot qui a besoin de bit 1 pour rester en vie car celle si font passer le courant si tu recupaire trop de bit 0 tu finis a cours d'énergie et tu meurs 

## les class 

### Ordinateur

* **perdreVie** 
retire le nombre de vie indiquer
* **ajoutScore**
ajoute au score le nombre indiquer
* **updateAffichage**
prend en parametre un nombre (les coordonnée de la souris en theorie)
il modifie l'atribut position_x avec le nombre entre
et met ajour l'affichage
* **initialisationAffichage**
cree le robot (panier) dans le dom afin de l'afficher

### Bit

### Reste a faire 
* coder les colisions ( bas de l'ecran et robot (panier) ) 
* faire supprimer les bit lorsque elles subbisent une collision
* ajouter le score si collision avec robot (panier) 
* faire afficher le score en binnaire en hexadecimale et en decimale
* faire la gestion des vie et leur affichage
